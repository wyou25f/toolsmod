A lightweight, server-side friendly utility designed to break the vanilla boundaries of Minecraft. It introduces the powerful `/tools` command, giving you complete control over item customization and server adjustments without bloating your game.

---

## English 🇬🇧

### 👥 Who is this mod for?
* **Server Administrators:** Perfect for creating unique event rewards, custom boss drops, or high-tier gear for players.
* **Content Creators & Mapmakers:** Easily gear up for showcases, testing, or building custom RPG maps with overpowered weapons.
* **Survival Lovers:** Want to have some chaotic fun with friends? Break the limits and see what Sharpness X can do!

### ✨ Main Features
* **Bypass Limits:** Go beyond vanilla restrictions! Enchant any item up to level 10 (e.g., `Sharpness X`, `Efficiency X`) instantly via commands.
* **Multi-language Support:** Built-in localization. Seamlessly switch languages on the fly.

### 📜 Command Syntax
* `/tools enchant [enchantment_id] [level]` — Enchants the item in your hand (e.g., `/tools enchant sharpness 10`).
* `/tools lang [en/ru]` — Instantly changes the mod's display language.

### 🚀 Future Roadmap (Where we are heading)
We want to expand this mod into an ultimate server administration toolkit. Upcoming features:
* Support for enchantments above level 10 (up to level 32767).
* Custom item renaming and lore editing via `/tools` subcommands.
* A simple config file to set max custom limits for survival mode.

## Русский 🇷🇺

### 👥 Кому полезен этот мод?
* **Администраторам серверов:** Идеально подходит для создания уникальных наград за ивенты, кастомного лута с боссов или выдачи админ-снаряжения.
* **Создателям контента и карт:** Быстро создавайте кастомное оружие для видеороликов, тестов или прохождения карт с элементами RPG.
* **Любителям веселья:** Хотите устроить хаос с друзьями? Снимите ограничения и посмотрите, на что способна Острота X!

### ✨ Основные возможности
* **Обход лимитов:** Забудьте о ванильных ограничениях! Зачаровывайте любые предметы до 10 уровня (например, `Острота X` или `Эффективность X`) одной командой.
* **Два языка «из коробки»:** Полная встроенная локализация с удобным переключением.

### 📜 Синтаксис команд
* `/tools enchant [id_зачарования] [уровень]` — Зачаровать предмет в руке (пример: `/tools enchant sharpness 10`).
* `/tools lang [en/ru]` — Моментально меняет язык мода.

### 🚀 Планы на будущее (Вектор развития)
Мы хотим превратить этот мод в ультимативный пак инструментов для управления сервером. В следующих обновлениях:
* Поддержка уровней чар выше 10 (вплоть до 32767).
* Изменение названий предметов и добавление описаний (Lore) через подкоманды `/tools`.
* Удобный конфиг-файл для ограничения максимального уровня чар в выживании.
